Esse projeto te ensina a mexer no Git

comando: git init para iniciar uma branch master vazia;

git add Readme.md (um único arquivo) git add . (tudo) : manda os arquivos para a área de standing (aguardando commit);

git status: mostra as mudanças a serem commitadas;

git commit -m: subindo arquivo para repositório;

git branch -M "main": renomeando o nome da branch para main;

git remote add origin: subir projeto para o github

git push -u origin main: para subir o push para o github
